# Recipes

Personal recipe collection.

---

## Recipe Format

Each recipe should follow this structure:

```markdown
---
type: recipe
tags: [protein, quick, meal-prep, etc.]
servings: X
prep_time: X min
cook_time: X min
source: URL or "original"
---

# Recipe Title

Brief description of the dish.

## Ingredients

- 1 cup ingredient one
- 2 tbsp ingredient two
- etc.

## Instructions

1. First step
2. Second step
3. etc.

## Notes

Optional notes, variations, tips, or lessons learned.
```

---

## Conventions

- **Filename:** `recipe-name.md` (lowercase, hyphenated)
- **Measurements:** Include both imperial and metric where helpful
- **Tags:** Use consistent tags for filtering:
  - `protein` - high protein recipes
  - `quick` - under 30 min total
  - `meal-prep` - good for batch cooking
  - `breakfast`, `lunch`, `dinner`, `snack`, `dessert`
  - `vegetarian`, `vegan`, `gluten-free` as applicable
- **Source:** Link to original if adapted, or "original" if yours
