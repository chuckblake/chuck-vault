# Decisions

Making and documenting important decisions across all life areas.

## Key Files

| File | Purpose |
|------|---------|
| `frameworks.md` | Decision-making frameworks and tools |
| `log.md` | Record of significant decisions |

## When to Document

Document decisions that are:
- Irreversible or costly to reverse
- Involve significant resources or commitment
- Set important precedents
- You might want to reference later

## Quick Reference

**Type 1 (One-way door):** Irreversible, go slow, analyze carefully
**Type 2 (Two-way door):** Reversible, go fast, delegate when possible
