# Templates

Reusable templates for structured documents.

## Available Templates

| Template | Purpose | Use When |
|----------|---------|----------|
| `hiring-template.md` | Hiring evaluation | Evaluating candidates |
| `performance-review-template.md` | Performance reviews | Review cycles |
| `project-template.md` | Project kickoff | Starting new projects |
| `article-template.md` | Article notes | Capturing article takeaways |
| `book-template.md` | Book notes | Capturing book notes |
| `1on1-template.md` | 1:1 meetings | Preparing for/documenting 1:1s |

## Using Templates

Copy the template to the appropriate location and fill it in:

```bash
cp templates/project-template.md projects/my-new-project/README.md
```

Or ask Claude to create a new file from a template.

## Creating New Templates

If you find yourself repeatedly creating similar documents, make a template:
1. Identify the repeating structure
2. Create a template with placeholders
3. Add it here with a description
