# 1:1 Prep

Help me prepare for a 1:1 meeting.

## Instructions for Claude

1. Ask who the 1:1 is with
2. Read `@me/people.md` for context on that person
3. Check for previous 1:1 notes in `areas/work/team/`
4. Reference recent journal entries for context

## Conversation Flow

### 1. Context

Ask me:
- Who is this 1:1 with?
- When is it?
- Any specific context or reason for this meeting?

### 2. Review History

Look for:
- Previous 1:1 notes with this person
- Any pending action items
- Recent mentions in journal entries

### 3. Build Agenda

Help identify:
- Follow-ups from last time
- Topics they might want to discuss
- Topics I need to bring up
- Feedback to give or receive
- Development/growth conversations

### 4. Prepare

Help me think through:
- Any difficult conversations needed?
- How are they doing overall?
- What support do they need?

## Output

Create or update 1:1 notes file:
- `areas/work/team/[name]-1on1s.md`

Use template from `@templates/1on1-template.md`
